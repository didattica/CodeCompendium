**Prompt migliorato:**

Devo preparare un corso basato su materiale Cisco già esistente, che include slide didattiche.

### Obiettivo

Voglio che tu analizzi:

1. Le slide del corso Cisco che ti fornirò
2. Il file che ho già preparato

### Cosa devi fare

* Migliorare il mio file aggiornandolo sulla base dei contenuti delle slide
* Suggerire come spiegare i concetti in modo:

  * più semplice e chiaro
  * adatto a studenti (contesto scolastico)
  * mantenendo però accuratezza scientifica
* Quando utile, usa:

  * metafore
  * esempi concreti
  * analogie intuitive

### Output richiesto

* Formato: **Markdown (.md)**
* Struttura consigliata:

  * Spiegazione semplificata dei concetti
  * Eventuali metafore/analogie
  * Suggerimenti didattici (come presentarlo a lezione)
  * Revisione/miglioramento del mio file originale

### Input

* Il mio file da aggiornare
* Le slide del corso Cisco
