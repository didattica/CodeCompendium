# 🧠 ESERCIZIO 4

## Dato:

```
192.168.1.0/24
```

## Richiesta:

Dividere in **8 sottoreti** e indicare:

* Indirizzo di rete
* Broadcast
* Range host

---

## 🔹 1️⃣ Calcolo bit da prendere

```
2^n ≥ 8
```

```
2^3 = 8
```

👉 n = 3 bit

---

## 🔹 2️⃣ Nuovo CIDR

```
/24 + 3 = /27
```

Subnet mask:

```
255.255.255.224
```

---

## 🔹 3️⃣ Calcolo incremento

```
256 - 224 = 32
```

---

## 🔹 4️⃣ Sottoreti

| ------ | ---------------- | ------------- | ----------- |

# 🧠 ESERCIZIO 5

## Dato:

```
172.16.0.0/22
```

## Richiesta:

Dividere in **4 sottoreti**.

---

## 🔹 1️⃣ Analisi iniziale

/22 significa:

```
32 - 22 = 10 bit host
```

Totale indirizzi:

```
2^10 = 1024
```

Range totale:

```
172.16.0.0 – 172.16.3.255
```

---

## 🔹 2️⃣ Bit necessari

```
2^2 = 4
```

n = 2

---

## 🔹 3️⃣ Nuovo CIDR

```
/22 + 2 = /24
```

Subnet mask:

```
255.255.255.0
```

---

## 🔹 4️⃣ Incremento

```
256 - 0 = 256
```

---

## 🔹 5️⃣ Sottoreti

| Subnet | Network       | Broadcast    | Host      |
| ------ | ------------- | ------------ | --------- |
| 1      | 172.16.0.0/24 | 172.16.0.255 | .1 – .254 |
| 2      | 172.16.1.0/24 | 172.16.1.255 | .1 – .254 |
| 3      | 172.16.2.0/24 | 172.16.2.255 | .1 – .254 |
| 4      | 172.16.3.0/24 | 172.16.3.255 | .1 – .254 |
