
# Esercizi CIDR – Network & Broadcast

---

## Esercizio 1

**Dato:**

IP: 192.168.10.130
CIDR: /25

### 1️⃣ Bit rete / host

/25 → 25 bit rete, 7 bit host

Subnet mask:

```
11111111.11111111.11111111.10000000
=
255.255.255.128
```

---

### 2️⃣ IP in binario

```
192.168.10.130 =
11000000.10101000.00001010.10000010
```

Mask:

```
11111111.11111111.11111111.10000000
```

---

### 3️⃣ AND bit a bit

```
11000000.10101000.00001010.10000010
11111111.11111111.11111111.10000000
=
11000000.10101000.00001010.10000000
```

➡️ Network Address:

```
192.168.10.128
```

---

### 4️⃣ Broadcast

Host bit = 7 → tutti a 1

```
10000000 → 11111111
```

➡️ Broadcast:

```
192.168.10.255
```
